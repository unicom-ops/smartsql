# 功能点介绍



**SmartSQL**是一款支持动态的智能化SQL语句审核平台。提供动态执行分析检查与SQL文本分析等多种功能。



### 支撑的数据类型

数据库类型为MySQL、Ali-RDS、Ali-DRDS、Oracle


### 主要功能：

* SQL审核

  * 提交SQL待审核工单
    * 支持手工添加单条SQL
    * 支持批量导入
  * SQL文本分析
  * SQL执行计划分析
  * 给出结果评分，生成审核报告

* 数据库管理

  * 数据库统计分析
  * 数据库增删改操作

* 规则管理

  * 审核规则统计分析
  * 审核规则个性化修改

* 审核统计

* 审核历史

  * 审核工单统计分析
  * 审核工单结果详情

* 系统管理

  * 配置策略清理
